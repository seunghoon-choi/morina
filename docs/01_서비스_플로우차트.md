## 3️⃣ 세무 밀착관리 (유료) 상세

```mermaid
flowchart TD
    Start([밀착관리 시작]) --> PayCheck{구독 상태?}
    PayCheck -->|미구독| PayWall[결제 화면<br/>월 9900원]
    PayWall --> Pay[간편결제]
    Pay --> Step1
    PayCheck -->|구독중| Step1

    Step1[STEP 1: 수입 확인] --> PrevRevenue[직전연도 매출 자동 로드]
    PrevRevenue --> ThisRevenue[올해 예상 매출<br/>편집 가능]
    ThisRevenue --> ExcelUpload[홈택스 매입세금계산서<br/>엑셀 업로드]
    ExcelUpload --> Parse[매입 엑셀 파싱]

    Parse --> Step2[STEP 2: 경비 자동분류]
    Step2 --> AutoCategory[세비가 분류<br/>광고비·임대료·자산구입 등]

    Manual --> ManualItems{경비 항목}
    ManualItems --> M1[인건비]
    ManualItems --> M2[접대비]
    ManualItems --> M3[광고선전비]

    ManualItems --> M5[경조사비]
    ManualItems --> M6[카드경비]

    M1 --> Calc[진단 시작]
    M2 --> Calc
    M3 --> Calc
    M4 --> Calc
    M5 --> Calc
    M6 --> Calc

    Calc --> Step4[STEP 4: 밀착관리 리포트]

    Final -->|리포트 보관| Vault[신고서 보관함<br/>자동 저장]
```

## 4️⃣ 인건비 등록 → 원천세 신고 가이드 (핵심 하이브리드 UX)

```mermaid
flowchart TD
    Start([원천세 신고하기]) --> Manage[인건비 관리 메인]
    Manage --> Register[인건비 등록 버튼]

    Register --> TypeSelect{직원 구분}
    TypeSelect -->|일용직| Daily[일당지급]
    TypeSelect -->|정직원| Regular[원천징수/상여처리대상자]

    Daily --> CollectMethod{신분증 수집 방법}
    Regular --> CollectMethod

    CollectMethod -->|카톡 링크| KakaoSend[팝빌 API<br/>알림톡 발송]
    CollectMethod -->|직접 입력| DirectInput[사장님이 직접 입력]

    KakaoSend --> EmpLink[직원용 링크 페이지]
    EmpLink --> Consent[개인정보 수집 동의]
    Consent --> EmpInput[이름/주민번호/연락처 입력]
    EmpInput --> Validate[체크섬 검증]
    Validate --> Encrypt[KMS 암호화 저장]

    DirectInput --> Validate

    Encrypt --> Received[수거된 신분증 목록]
    Received --> Vault[본인보관함 자동저장]

    Vault --> Calc[원천세 자동 계산]
    Calc --> Simple[간이세액표 기반]
    Simple --> Amount[이번달 원천세<br/>예: 220,140원]

    Amount --> Detail[직원별 상세 내역]
    Detail --> HometaxStart[홈택스 신고가이드 진입]

    HometaxStart --> HybridConcept[하이브리드 UX 개념]
    HybridConcept --> Mobile[📱 모바일 바이택스<br/>값 제공/확인]
    HybridConcept --> PC[💻 PC 홈택스<br/>실제 신고]

    Mobile --> StepByStep[단계별 가이드]
    StepByStep --> S1[1. PC에서 홈택스 접속]
    StepByStep --> S2[2. 공동인증/간편인증 로그인]
    StepByStep --> S3[3. 신고/납부 → 원천세]
    StepByStep --> S4[4. 이 화면 내용 그대로 입력]

    S4 --> InputGuide[홈택스 입력 가이드 화면]
    InputGuide --> I1[귀속연월: 2026년 1월]
    InputGuide --> I2[지급연월: 2026년 1월]
    InputGuide --> I3[인원: 일용직 1 + 상용직 1 = 2]
    InputGuide --> I4[총지급액]
    InputGuide --> I5[소득세]

    InputGuide --> UserAction{액션}
    UserAction -->|카톡 전송| ShareMe[내게 카톡 보내기]
    UserAction -->|PDF| PDFSave[PDF 저장/공유]
    UserAction -->|세무사 검토| Reserve[검토 예약하기]

    Reserve --> Advisor[세무사 상담 예약]

    style HybridConcept fill:#5B6CF9,color:#fff
    style Calc fill:#90EE90
    style Amount fill:#FFD700
```

## 📊 화면 간 이동 매트릭스

| 출발 화면 | 도착 화면 | 트리거 |
|---|---|---|
| 홈 | 기초검진 | "1분 세무진단" 버튼 |
| 기초검진 리포트 | 밀착관리 | "세무 밀착관리 시작" (구독 유도) |
| 밀착관리 리포트 | 세무사 예약 | "복식부기의무자 세무사 상담" |
| 인건비 등록 | 원천세 계산 | "저장" → 자동 계산 |
| 원천세 계산 | 홈택스 가이드 | "홈택스 원천세 신고가이드" |
| 홈택스 가이드 | 세무사 예약 | "검토 예약하기" |
| 모든 리포트 | 보관함 | 자동 저장 (구독자) |
| 캘린더 | 기초검진 | 신고 D-day 이벤트 클릭 |

## 🎯 핵심 전환 포인트 (Conversion Points)

```mermaid
flowchart LR
    Visitor[방문자] -->|가입| Free[무료 유저]
    Free -->|1분 검진 완료| FreeActive[Active 무료 유저]
    FreeActive -->|85점 리포트 충격| Curious[관심 유저]
    Curious -->|밀착관리 유혹| Trial[결제 고려]
    Trial -->|결제 완료| Sub[구독 유저]

    Sub -->|복식부기의무자| Premium[세무사 연결]
    Premium -->|상담 완료| SuccessCase[성공 케이스]

    style Free fill:#E3F2FD
    style Sub fill:#FFD700
    style SuccessCase fill:#90EE90
```

**핵심 KPI 측정 지점**:
1. `Visitor → Free` — 회원가입 전환율
2. `Free → FreeActive` — 1분 검진 완료율
3. `FreeActive → Sub` — 유료 전환율 (가장 중요)
4. `Sub → Premium` — 세무사 연결 전환율 (부가 수익)
